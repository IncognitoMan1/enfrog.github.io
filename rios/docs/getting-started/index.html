Getting Started

RIOS for OS Devs:

Setting up RIOS on your gadget

  • Download rios.lua from any example file or an empty one at the official RIOS github page.
    https://github.com/VincentFoulon80/retro-input-output-system
  • If you took the empty file, fill the functions with missing functionality. (See --todo) in the files functions (rios.lua)
  • If you took any of the example file, Set up the two device lists located in the middle of the rios file, below the constants
  • On the CPU file of your gadget, require rios.lua in a variable

    • local rios = require("rios.lua")

  • (OPTIONAL) remove access to gdt just after requiring RIOS

    • gdt = nil

Running apps using RIOS

  • Require the apps you want to run, and use the registerApp function to register it to RIOS
  • In the update() function, run:

    • rios.runApps(rios)
